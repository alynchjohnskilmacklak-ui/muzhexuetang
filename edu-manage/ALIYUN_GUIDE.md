# 阿里云部署指南

## 第一步：购买服务器

1. 打开 https://ecs.console.aliyun.com → 登录
2. 点 **创建实例**
3. 选择：
   - **地区**：离你最近的（如杭州/上海）
   - **镜像**：Ubuntu 22.04
   - **规格**：2核2G（¥68/月，够用了）
   - **系统盘**：40GB
   - **带宽**：按量计费，5Mbps
4. 创建 → 付钱 → 等待启动

## 第二步：开放端口

1. 进入实例详情 → **安全组** → **配置规则**
2. 添加规则：
   | 端口 | 协议 | 来源 |
   |------|------|------|
   | 3000 | TCP | 0.0.0.0/0 |
   | 22 | TCP | 0.0.0.0/0 |
   | 5432 | TCP | 127.0.0.1/32 |

## 第三步：连接服务器

打开终端（PowerShell），用服务器公网IP连接：
```
ssh root@你的公网IP
```

## 第四步：运行部署脚本

登录服务器后：
```bash
cd /opt/muzhexuetang
bash deploy.sh
```

## 第五步：访问

浏览器打开：`http://你的公网IP:3000`

## 可选：绑定域名

1. 阿里云域名解析 → 添加 A 记录 → 指向服务器 IP
2. 安装 nginx 反向代理
3. 配置 HTTPS（Let's Encrypt 免费证书）
